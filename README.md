# Brain-Tumor-3D-Segmentation-with-MONAI
High-performance 3D Brain Tumor Segmentation pipeline using MONAI and PyTorch. Processes multi-modal MRI (FLAIR, T1, T1ce, T2) via volumetric UNet to segment TC, WT, and ET regions. Features automated 3D transforms, Dice/HD95 metrics, and Kaggle-ready checkpoint recovery. Trained on the MSD Task 01 dataset using NIfTI files for clinical accuracy.
